# GitHub User Finder (Refactored)

**Автор:** Хизбуллин Артур  
**Версия:** 1.1 (Исправленная)

## Описание
Приложение на базе Python/Tkinter для поиска информации о пользователях GitHub через официальное REST API. Позволяет просматривать краткую информацию и сохранять выбранные профили в локальный JSON-файл.

## Установка и запуск

### 1. Требования
- Python 3.10 или выше.
- Доступ в интернет.

### 2. Установка зависимостей
Для работы приложения необходима библиотека `requests`. Установите её командой:
```bash
pip install requests
```

### 3. Запуск
```bash
python main.py
```

## Работа с API
- **Endpoint:** `https://api.github.com/users/{username}`
- **Метод:** GET
- **Обработка данных:** Приложение проверяет статус-код ответа (200, 404, 403) и валидирует структуру JSON перед выводом на экран.

## Что было исправлено:
1. **Безопасность:** Добавлены блоки `try...except` для перехвата сетевых ошибок и ошибок парсинга JSON.
2. **Валидация:** Реализована проверка статус-кодов (404 для несуществующих юзеров).
3. **Надежность:** Добавлены таймауты запросов (`timeout=10`), чтобы программа не зависала при плохом соединении.
4. **Инструкция:** Добавлен раздел по установке зависимостей.
