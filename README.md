# GitHub User Finder

**Автор:** [Ваше Имя и Фамилия]

### Описание
Приложение для быстрого поиска профилей пользователей GitHub и сохранения их в локальный список "Избранное". Данные сохраняются в формате JSON.

### Как использовать API
Приложение использует публичное [GitHub REST API](https://github.com). 
- Запрос выполняется по адресу: `https://github.com{username}`
- Лимит для неавторизованных запросов: 60 в час.

### Примеры использования
1. Введите в поле поиска `torvalds`.
2. Нажмите кнопку "Найти".
3. Нажмите "Добавить в избранное", чтобы сохранить пользователя в файл `favorites.json`.
